# HTML-Only Personal Website Project

## Project Overview
[cite_start]This project is a fully structured, multi-page website built strictly using **pure HTML**[cite: 8, 10]. [cite_start]The objective was to implement core HTML elements and semantic layout tags without the use of CSS or JavaScript[cite: 5, 6, 7].

## Features
- [cite_start]**Zero CSS/JS**: Every aspect of the site is handled by native HTML elements[cite: 6, 7].
- [cite_start]**Semantic Structure**: Uses `<header>`, `<nav>`, `<main>`, `<article>`, and `<footer>` for layout[cite: 22].
- [cite_start]**Full Form Integration**: A comprehensive registration form with various input types[cite: 53].
- [cite_start]**Multimedia**: Embedded images, audio, video, and a Google Map[cite: 51, 54].
- [cite_start]**Data Display**: Structured tables and lists for service descriptions[cite: 46, 47].
